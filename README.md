# Scripts

Random collection of useful scripts

## Usage

### AWS

#### Install/Update the AWS CLI

Run the following terminal command

```bash
curl -sSL https://raw.githubusercontent.com/Colin23/scripts/refs/heads/main/aws/install-or-update-aws-cli | bash
```

#### Uninstall the AWS CLI

```bash
curl -sSL https://raw.githubusercontent.com/Colin23/scripts/refs/heads/main/aws/uninstall-aws-cli | bash
```
